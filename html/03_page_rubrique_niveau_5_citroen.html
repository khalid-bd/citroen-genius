Installer les plaques sur les véhicules exposés dans le showroom Dès qu’un véhicule est dans le showroom
Ajouter à mon calendrier
Vendeurs
Préparateurs

Description

Dès qu’un véhicule est exposé dans le hall, il est indispensable qu’il soit identifié par des plaques showroom. Chaque véhicule a une paire de plaques (avant et arrière) indiquant son nom (exemple : Citroën C4 Picasso).

Exemple : CITOËN C4 PICASSO

Les plaques se commandent sur la Boutique. Ne pas oublier de commander également des velcros (boite de 200) pour fixer les plaques sur les véhicules.
